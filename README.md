# wine-quality-predictive-regression
Modelo de regresión para la predicción de calidad del vino basado en características fisicoquímicas. Comparativa de algoritmos (Random Forest, XGBoost, KNN) con optimización de hiperparámetros (Grid Search) y enfoque en Model Governance y calidad del dato. Proyecto del Máster en Business Analytics de la UAM (Abril 2026).
